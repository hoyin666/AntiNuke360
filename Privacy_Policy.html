隱私政策

最後更新：2025年11月18日

1. 簡介

AntiNuke360（以下簡稱「本機器人」）非常重視您的隱私。本隱私政策說明本機器人如何收集、使用、保護和披露您的信息。

2. 我們收集的信息

本機器人在運作過程中可能會收集以下信息：

  • 伺服器 ID 和信息：用於識別和管理伺服器，以及儲存設定
  • 用戶 ID 和用戶名：用於黑名單和白名單管理
  • 訊息內容：僅用於反被盜帳偵測功能
  • 成員信息：用於安全掃描和驗證
  • 權限信息：用於權限檢查和防護
  • 操作日誌：記錄所有防護操作和管理員指令

3. 信息的使用

我們使用收集的信息來：

  • 提供和改進反核打擊防護功能
  • 維護和更新全域黑名單和白名單系統
  • 執行伺服器安全掃描和偵測
  • 發送重要安全警報和通知
  • 調查和防止不當使用
  • 遵守法律和監管要求

4. 反被盜帳偵測

本機器人的反被盜帳偵測功能會讀取訊息內容以識別可疑的發送模式。具體來說：

  • 監控 5 秒內在不同頻道發送相同內容的行為
  • 檢測到疑似被盜帳號時自動採取行動（刪除訊息和踢出用戶）
  • 向被害人發送恢復邀請連結
  • 訊息內容不會被儲存或用於其他目的

5. 信息安全

我們採取適當的技術和組織措施來保護您的信息免受未經授權的訪問、修改、披露或破壞。然而，沒有傳輸方法可以 100% 安全。

6. 信息保留

我們會保留必要的信息以提供服務。快照數據保留 72 小時後自動刪除。操作日誌和黑名單信息會根據需要保留。

7. 信息共享

我們不會與第三方共享您的個人信息，除非：

  • 您明確同意
  • 法律要求
  • 保護本機器人或他人的權利、隱私或安全

8. 兒童隱私

本機器人不有意收集 13 歲以下兒童的信息。如果我們發現我們收集了該年齡段兒童的信息，我們將立即刪除該信息。

9. Discord 集成

本機器人通過 Discord Bot API 運作。Discord 的數據實踐受其隱私政策約束。我們不控制 Discord 對信息的收集或使用。

10. 隱私設置

伺服器管理員可以：

  • 配置記錄頻道以控制日誌位置
  • 使用 `/toggle-anti-hijack` 開啟或關閉反被盜帳功能
  • 通過移除機器人的權限來完全停用本機器人

11. 隱私政策的更改

我們可能會不時更新本隱私政策。我們會在本網頁上發佈任何重大變更。持續使用本機器人表示您同意更新後的隱私政策。

12. 聯繫我們

如果您對本隱私政策有任何疑問或關注，請通過以下方式聯繫我們：

  • 在 GitHub 上提交問題：https://github.com/hoyin666/AntiNuke360
  • 通過 Discord 聯繫開發者
重要提示：本隱私政策不是法律文件，而是對本機器人數據實踐的說明。由於本機器人是開源的，任何人都可以查看源代碼以了解確切的數據處理方式。

13. 開源透明度

AntiNuke360 是完全開源的。我們的源代碼在 GitHub 上公開可用，允許任何人審查我們如何處理您的數據。這提供了最高級別的透明度。

14. 數據所有權

您保留您所有內容的所有權。我們只是為了提供服務而處理數據。您隨時可以要求刪除數據。
