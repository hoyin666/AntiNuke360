介紹

AntiNuke360 是一個強大的 Discord 伺服器防護機器人，專門針對核打擊 (Nuke) 攻擊進行防禦。我不像安全衛士360那麼流氓，但我至少能防止99.99%的核打擊。

功能

自動反 Nuke 防護

持續監控大量刪除頻道、踢出成員等異常行為，在 10 秒內自動識別並阻止攻擊，防止伺服器被一鍵炸毀。

全域黑名單系統

維護全域黑名單，記錄已知惡意機器人與攻擊者，新成員加入時即時比對並強制移除。

本地伺服器白名單

每個伺服器都可獨立管理白名單，可信任的機器人不會被防護系統干擾。

全域白名單系統

由開發者維護的全域白名單，允許指定機器人在所有伺服器中豁免防護。

稽核日誌監控

監控頻道、角色、成員和 Webhook 的建立與刪除，追蹤每一次敏感操作。

權限錯誤監控

自動偵測權限不足情況，當問題累積到一定次數會通知伺服器擁有者，必要時自動離開伺服器。

成員加入掃描

新成員加入時自動檢查黑名單和白名單，將風險降到最低。

自訂狀態文字

內建多條狀態文字，機器人會自動輪替顯示，提高可見度與辨識度。

安裝

先決條件

  • Python 3.8 或更高版本
  • discord.py
  • python-dotenv

安裝步驟

  1. 複製倉庫或下載程式碼。
  2. 建立 .env 檔案並設置 Discord Bot Token： 
    DISCORD_TOKEN=your_discord_bot_token_here
  3. 安裝依賴： 
    pip install -r requirements.txt
  4. 啟動機器人： 
    python AntiNuke360_v1.0.py

Discord Bot 設置

  1. 前往 Discord Developer Portal (https://discord.com/developers/applications)。
  2. 建立新 Application，並在 Bot 分頁建立機器人。
  3. 複製 Bot Token 到 .env 檔案。
  4. 在 OAuth2 的 URL Generator 中勾選所需權限：
    • administrator（或個別勾選 ban members、kick members、manage channels、manage roles、manage webhooks）。
  5. 使用生成的邀請連結將機器人加入伺服器。

指令

/status

查看 AntiNuke360 的當前運行狀態、防護參數與黑名單統計。

/scan-blacklist

掃描伺服器內所有成員，並停權全域黑名單中的成員。需要管理員權限。

/add-server-white [ID]

將指定機器人加入本伺服器白名單。需要管理員權限。

/remove-server-white [ID]

從本伺服器白名單移除指定機器人。需要管理員權限。

/server-whitelist

查看本伺服器目前的白名單成員列表。需要管理員權限。

開發者指令

以下指令僅限開發者使用。

/add-black [ID] [原因]

將機器人加入全域黑名單，並在所有伺服器中掃描與停權。

/remove-black [ID]

從全域黑名單移除指定機器人。

/add-white [ID] [原因]

將機器人加入全域白名單。

/remove-white [ID]

從全域白名單移除指定機器人。

/blacklist

查看全域黑名單，顯示前若干筆記錄。

/whitelist-list

查看全域白名單，顯示前若干筆記錄。

/scan-all-guilds

在所有伺服器中掃描並停權全域黑名單成員。

常見問題

AntiNuke360 是完全免費嗎？

是。AntiNuke360 永久免費，亦永久開源，不會收取任何費用。

可以修改並重新發佈 AntiNuke360 嗎？

可以。本專案採用 MIT License，只要保留原始許可聲明即可。

防護參數可以調整嗎？

目前防護參數固定為 10 秒內最多 7 次敏感操作，無法調整，以確保風險控制在預期範圍內。

機器人需要哪些權限？

建議給予 administrator 權限；若不方便，可至少給予 ban members、kick members、manage channels、manage roles、manage webhooks。

支援多少伺服器？

理論上不限制伺服器數量，實際上取決於你所使用的託管環境資源。
