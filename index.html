介紹

AntiNuke360 是一個強大的 Discord 伺服器防護機器人，專門針對核打擊 (Nuke) 攻擊進行防禦。我不像安全衛士360那麼流氓，但我至少能防止99.99%的核打擊。

功能

自動反 Nuke 防護

持續監控大量刪除頻道、踢出成員等異常行為，在 10 秒內自動識別並阻止攻擊。

伺服器快照系統v1.1+

自動建立伺服器結構快照，遭受攻擊後可完全還原。快照保留 72 小時。

三層白名單系統v1.2 安全修復

防踢白名單、臨時白名單、永久白名單，區分管理員和擁有者權限，防止濫用。

全域黑名單系統

維護全域黑名單，記錄已知惡意機器人與攻擊者，新成員加入時即時比對並強制移除。

全域白名單系統

由開發者維護的全域白名單，允許指定機器人在所有伺服器中豁免防護。

稽核日誌監控

監控頻道、角色、成員和 Webhook 的建立與刪除，追蹤每一次敏感操作。

權限錯誤監控

自動偵測權限不足情況，當問題累積會通知伺服器擁有者，必要時自動離開。

成員加入掃描

新成員加入時自動檢查黑名單和白名單，將風險降到最低。

記錄頻道系統v1.2 新增

管理員可設定記錄頻道，所有防護事件和白名單操作都會記錄。

Administrator 檢查v1.2.1 新增

自動檢查並確保具有完整權限，若無權限會通知管理員並自動離開。

安裝

先決條件

  • Python 3.8 或更高版本
  • discord.py
  • python-dotenv

安裝步驟

  1. 複製倉庫或下載程式碼。
  2. 建立 .env 檔案並設置 Discord Bot Token： 
    DISCORD_TOKEN=your_discord_bot_token_here
  3. 安裝依賴： 
    pip install -r requirements.txt
  4. 啟動機器人 (v1.2.1)： 
    python AntiNuke360_v1.2.1.py

Discord Bot 設置

  1. 前往 Discord Developer Portal。
  2. 建立新 Application，並在 Bot 分頁建立機器人。
  3. 複製 Bot Token 到 .env 檔案。
  4. 在 OAuth2 的 URL Generator 中勾選所需權限。
  5. 使用生成的邀請連結將機器人加入伺服器。

指令

/status

查看 AntiNuke360 的當前運行狀態、防護參數與統計。

/scan-blacklist

掃描伺服器內所有成員，並停權全域黑名單中的成員。需要管理員權限。

/restore-snapshot

手動還原伺服器最新的快照。還原所有角色、分類、頻道、權限設定。需要管理員權限。

/add-server-temp [ID]

將機器人加入臨時白名單 (1 小時自動過期)。需要管理員權限。

/remove-server-temp [ID]

從臨時白名單移除機器人。需要管理員權限。

/server-whitelist

查看本伺服器的白名單狀態 (防踢、臨時、永久)。需要管理員權限。

/set-log-channel [#channel]

設置本伺服器的記錄頻道。所有防護事件都會記錄在此。需要管理員權限。

伺服器擁有者指令

/add-server-anti-kick [ID]

將機器人加入防踢白名單。允許全域黑名單帳號加入但記錄監控。

/remove-server-anti-kick [ID]

從防踢白名單移除機器人。

/add-server-perm [ID]

將機器人加入永久白名單。對所有敏感操作完全豁免。

/remove-server-perm [ID]

從永久白名單移除機器人。

/remove-black [ID]

常見問題

v1.2.1 新增了什麼？

v1.2.1 新增了自動 Administrator 權限檢查。當機器人加入伺服器時，會檢查是否具有完整的 Administrator 權限。如果沒有，會自動通知伺服器擁有者和管理員，然後自動離開伺服器。這確保了機器人能夠正常提供完整的防護和還原功能。

機器人在我的伺服器中說無法運作怎麼辦？

這通常表示機器人缺少 Administrator 權限。請確保在邀請機器人時已授予 Administrator 權限。v1.2.1 會自動檢查並通知您。

v1.2 修復了什麼 0day 漏洞？

v1.2 修復了一個重要的安全漏洞：任何管理員都可以將任何機器人加入伺服器白名單，導致被白名單的機器人可以進行任何操作而不觸發警報。v1.2 實現了三層白名單系統，區分了管理員和擁有者的權限，防止濫用。

快照會保留多久？

每個伺服器的快照保留 72 小時（3 天）。超過 72 小時的快照會自動清理。

AntiNuke360 是完全免費嗎？

是。AntiNuke360 永久免費，亦永久開源，不會收取任何費用。
